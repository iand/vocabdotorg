Introduction

This vocabulary is an expression into RDF of the concepts and relations described in the IFLA report on the Functional Requirements for Bibliographic Records (FRBR).

This vocabulary is a work in progress, it includes RDF classes for the group 1, 2 and 3 entities described by the FRBR report and properties corresponding to the core relationships between those entities. It does not yet describe attributes of the entities. Where possible, appropriate relationships with other vocabularies are included in order to place this vocabulary in the context of existing RDF work.

Changes From Previous Version

  • Added frbr:Person class.
  • Changed documentation so that classes are listed in same order as original FRBR report.
  • frbr:ResponsibleEntity is now disjoint with frbr:Subject.
  • Added comments for each class.
  • Made frbr:Item a subclass of geo:SpatialThing
  • Made frbr:Place a subclass of geo:SpatialThing
  • Made frbr:Object a subclass of geo:SpatialThing
  • Made frbr:Concept a subclass of skos:Concept
  • Made frbr:Work a subclass of wordnet:Work~2
  • Made frbr:Event a subclass of owltime:Event

Class: Endeavour

An endeavour is any one of the FRBR group one entities. These represent the products of artistic or creative endeavour.

URI:http://purl.org/vocab/frbr/core#Endeavour
Label:endeavour
Disjoint withfrbr:ResponsibleEntity and frbr:Subject
Equivalent to(composite term, see schema)

Class: Work

This class corresponds to the FRBR entity 'Work' which is an abstract notion of an artistic or intellectual creation.

URI:http://purl.org/vocab/frbr/core#Work
Label:work
Subclass offrbr:Endeavour and wordnet:Work~2
Disjoint withfrbr:Expression, frbr:Manifestation and frbr:Item

Class: Expression

This class corresponds to the FRBR entity 'Expression' which is a realization of a single work usually in a physical form.

URI:http://purl.org/vocab/frbr/core#Expression
Label:expression
Subclass offrbr:Endeavour
Disjoint withfrbr:Work, frbr:Item and frbr:Manifestation

Class: Manifestation

This class corresponds to the FRBR entity 'Manifestation' which is the physical embodiment of one or more expressions.

URI:http://purl.org/vocab/frbr/core#Manifestation
Label:manifestation
Subclass offrbr:Endeavour
Disjoint withfrbr:Work, frbr:Expression and frbr:Item

Class: Item

This class corresponds to the FRBR entity 'Item' which is an exemplar of a single manifestation.

URI:http://purl.org/vocab/frbr/core#Item
Label:item
Subclass offrbr:Endeavour and geo:SpatialThing
Disjoint withfrbr:Work, frbr:Expression and frbr:Manifestation

Class: ResponsibleEntity

A responsible entity is any one of the FRBR group two entities. These represent those responsible for the creation, production, distribution or maintenance of artistic or creative content.

URI:http://purl.org/vocab/frbr/core#ResponsibleEntity
Label:responsible entity
Disjoint withfrbr:Endeavour and frbr:Subject
Equivalent to(composite term, see schema)

Class: Person

This class corresponds to the FRBR entity 'Person' which is a living or dead individual.

URI:http://purl.org/vocab/frbr/core#Person
Label:person
Disjoint withfrbr:Endeavour and frbr:Subject
Equivalent tofoaf:Person

Class: CorporateBody

This class corresponds to the FRBR entity 'Corporate Body' which is an organization or group of individuals and/or organizations.

URI:http://purl.org/vocab/frbr/core#CorporateBody
Label:corporate body
Subclass offrbr:ResponsibleEntity
Equivalent to(composite term, see schema)

Class: Subject

A subject is any one of the FRBR group three entities. These represent things that may serve as the subjects of works

URI:http://purl.org/vocab/frbr/core#Subject
Label:subject
Disjoint withfrbr:ResponsibleEntity and frbr:Endeavour
Equivalent to(composite term, see schema)

Class: Concept

This class corresponds to the FRBR entity 'Concept' which is an abstract idea.

URI:http://purl.org/vocab/frbr/core#Concept
Label:concept
Subclass offrbr:Subject and skos:Concept
Disjoint withfrbr:Object, frbr:Event and frbr:Place

Class: Object

This class corresponds to the FRBR entity 'Object' which is a tangible or material thing.

URI:http://purl.org/vocab/frbr/core#Object
Label:object
Subclass offrbr:Subject and geo:SpatialThing
Disjoint withfrbr:Event, frbr:Place and frbr:Concept

Class: Event

This class corresponds to the FRBR entity 'Event' which is an action or an occurence.

URI:http://purl.org/vocab/frbr/core#Event
Label:event
Subclass offrbr:Subject and owltime:Event
Disjoint withfrbr:Place, frbr:Object and frbr:Concept

Class: Place

This class corresponds to the FRBR entity 'Place' which is a location.

URI:http://purl.org/vocab/frbr/core#Place
Label:place
Subclass offrbr:Subject and geo:SpatialThing
Disjoint withfrbr:Event, frbr:Object and frbr:Concept
