Academic Institution Internal Structure Ontology Roles (AIISO Roles)

Other formats: RDF/XML

Creators
Rob Styles
Chris Wallace
The AIISO Roles schema provides common roles found in academic institutions in higher education. It is intended for use with Participation (http://purl.org/vocab/participation/schema), FOAF (http://xmlns.com/foaf/0.1/) and AIISO (http://purl.org/vocab/aiiso/schema) to describe the roles that people play within an institution. [en]

Namespace

The URI for this vocabulary is

http://purl.org/vocab/aiiso-roles/schema#

When abbreviating terms the suggested prefix is aiiso-roles

Each class or property in the vocabulary has a URI constructed by appending a term name to the vocabulary URI. For example:

http://purl.org/vocab/aiiso-roles/schema#Deputy_Chair

Academic Institution Internal Structure Ontology Roles (AIISO Roles)
The AIISO Roles schema provides common roles found in academic institutions in higher education. It is intended for use with Participation (http://purl.org/vocab/participation/schema), FOAF (http://xmlns.com/foaf/0.1/) and AIISO (http://purl.org/vocab/aiiso/schema) to describe the roles that people play within an institution.
